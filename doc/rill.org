** Stages

   - Prepare aggregates

     - Fetch secondary aggregates
   - Check primary aggregate version
   - Handle command

   - Commit events
   - Notify observers

     - Fetch secondary aggregates
     - Handle observation
     - Commit triggered events
     - Notify observers of triggered events
